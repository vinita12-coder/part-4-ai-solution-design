# Part 4: AI Solution Design for a Business Problem

## Domain
Finance — Loan Default Prediction

## Problem
Banks cannot efficiently identify high-risk loan applicants using manual
processes. This design proposes an AI-based binary classification solution
using a Feed-Forward Neural Network.

## Deliverables
- solution_report.md — Full 8-task written analysis
- diagrams/solution_architecture.png — System architecture diagram

## Summary
| Field | Details |
|---|---|
| AI Task Type | Binary Classification |
| Model | Feed-Forward Neural Network |
| Input Data | Customer financial and credit records |
| Output | Default Risk Score (0 = Safe, 1 = Default) |
| Business Impact | Faster approvals, reduced defaults |
